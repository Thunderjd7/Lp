Consider the following database:  
Employee (emp_no, name, skill, pay rate)  
Position (posting_no, skill)  
 
Insert one document at a time  
Insert Multiple documents using batch insert.  
Remove a document using $where.  
Update a document using $where.  
Upserting a document using save(). 



Step 1: Create and use the database

use CompanyDB


Step 2: Create Employee collection
1️⃣ Insert one document at a time

db.Employee.insertOne({ emp_no: 101, name: "Ravi", skill: "Java", pay_rate: 50000 })


2️⃣ Insert multiple documents using batch insert

db.Employee.insertMany([
  { emp_no: 102, name: "Mangesh", skill: "Python", pay_rate: 55000 },
  { emp_no: 103, name: "Akshay", skill: "C++", pay_rate: 48000 },
  { emp_no: 104, name: "Pooja", skill: "Web Development", pay_rate: 60000 },
  { emp_no: 105, name: "Sneha", skill: "Database", pay_rate: 52000 }
])


✅ Check inserted records:

db.Employee.find().pretty()


Step 3: Remove a document using $where
Remove employee whose pay_rate is less than 50000

db.Employee.remove({ $where: "this.pay_rate < 50000" })


Step 4: Update a document using $where
Increase pay rate by 10% for employees having “Python” skill

db.Employee.update(
  { $where: "this.skill == 'Python'" },
  { $inc: { pay_rate: 0.10 * 55000 } }
)


Step 5: Upserting (insert or update) a document using save()
If employee with emp_no:106 exists → update;
otherwise → insert new record.

db.Employee.save({ emp_no: 106, name: "Vikas", skill: "Cloud", pay_rate: 65000 })


Step 6: Create Position collection

db.Position.insertMany([
  { posting_no: "P01", skill: "Java" },
  { posting_no: "P02", skill: "Python" },
  { posting_no: "P03", skill: "Database" }
])


✅ Verification Queries

Display all employees:

db.Employee.find().pretty()


Display all positions:

db.Position.find().pretty()
